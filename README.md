# 🎬 Movie Ticket Booking System

A console-based Movie Ticket Booking System developed using C# and .NET. The application allows users to view movies, manage bookings, process payments, validate wallet balances, and maintain booking history using file handling.

## Features

* View Movies
* View Customers
* Book Tickets
* Wallet Balance Validation
* UPI & Card Payment
* Booking History
* File Handling
* Exception Handling

## Technologies Used

* C#
* .NET
* OOP Concepts
* File Handling
* Exception Handling

## Menu Options

1. View Movies
2. View Customers
3. Book Ticket
4. View Booking History
5. Exit

## Booking Flow

* Select Customer
* Select Movie
* Select Show
* Enter Ticket Count
* Calculate Total Amount
* Validate Wallet Balance
* Choose Payment Method (UPI/Card)
* Process Payment
* Save Booking Details
* Display Success Message
